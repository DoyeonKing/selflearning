## 生成二维码的步骤

### 方法1：使用在线工具批量生成

1. 打开 `tools/qrcode_generator.html`
2. 所有节点的二维码会自动显示
3. 点击"下载"按钮保存每个二维码

### 方法2：使用Python脚本批量生成

```bash
cd tools
python generate_qrcode.py
```

### 方法3：使用在线API直接生成

访问以下链接即可生成对应的二维码图片：

- HOSPITAL_NODE_1: https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=HOSPITAL_NODE_1
- HOSPITAL_NODE_2: https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=HOSPITAL_NODE_2
- HOSPITAL_NODE_3: https://api.qrserver.com/v1/create-qr-code/?size=500x500&data=HOSPITAL_NODE_3
- ... (以此类推)

## 上传二维码到系统

生成二维码图片后，使用以下API上传：

```bash
# 上传节点1的二维码
curl -X POST http://localhost:8080/api/qrcode/upload/1 \
  -F "file=@qrcode_node_1.png"
```

## 更新二维码状态

上传成功后，二维码状态会自动更新为 `ACTIVE`。
